Steps to run project 


step 1: create virtual environment using 
        python -m venv venv (in windows) 
        python3 -m venv venv (in linux / mac)
        
step 2: venv/scripts/activate (in windows)
        source venv/bin/activate (in linux / mac)



step 3: pip install -r requirements.txt 

step 4: py manage.py runserver
